Listas

Introdução

Se você tem uma lista de itens, armazená-los em variáveis individuais pode ficar assim:

fruta1 = "pera";
fruta2 = "pêssego";
fruta3 = "pitanga";

E se você não tem 3, mas 300 itens? A solução é o tipo de dados list do Python!

Uma list, como o nome indica, é uma variável que permite armazenar uma lista de valores, e não apenas um único valor.

Cada item da lista também pode ser chamado de elemento ou valor.

Ao invés de se referir a cada um dos itens por um nome diferente, você se refere a eles pelo nome da lista e os distingue por um índice, que é um número natural.

Também há o tipo de dados array, também chamado de vetor, que possui características semelhantes à listas, mas que não é tão versátil quanto listas.

Declarando listas

Você pode declarar uma lista com itens:

frutas=['abacate', 'banana', 'caqui', 'damasco', 'figo']

Ou declarar uma lista vazia para posterior adição de itens:

legumes=[]

Arrays em geral são homogêneos, podem conter apenas itens de um mesmo tipo. Por exemplo, você não pode ter um array que tenha um número e uma string. Mas listas em Python são heterogêneas, isto é, você pode colocar qualquer tipo de dados em uma mesma lista:

itens_diversos = ['uma palavra', 23, 9.5, True, ['uma', 'lista']]

Apesar de listas serem heterogêneas, na prática, elas acabam por armazenar apenas dados homogêneos.

Abstração

Uma lista é composta de vários itens. Pense numa lista como uma rua com várias casas. Os itens são moradores das casas da rua. E cada casa tem um endereço, um número, que na lista chamamos de índice ou posição.

Código Memória 
limeira = ["Cebolinha", "Mônica", "Cascão", "Magali"]
 
print("A rua Limeira possui quatro casas. \
Dentre os moradores, estão o Cebolinha, \
no endereço de número 0, \
e a Magali, no número 3.")

Mônica 1
Cascão 2
Magali 3

Manipulando listas

Para atribuir um item específico de uma lista a uma variável:

var = lista[i] //acessando o i-ésimo item da lista

Para atribuir um valor a um item específico de uma lista:

lista[i] = var //modificando o i-ésimo item da lista

O i, chamado de índice, ou posição, deve ser um número inteiro.

O primeiro item está no índice zero.

Para uma lista com N itens, o último item está no índice N-1.

Pode colocar qualquer expressão entre os colchetes, desde que seu resultado seja um número inteiro dentro do intervalo de índices válidos da lista.

Acessando itens

Considere o seguinte trecho de código:

qtd = 10  # comprimento da lista
nomes = ["maria", "ana", "francisca", "antonia", "adriana",
         "jose", "joao", "antonio", "francisco", "luiz"]

Você pode imprimir a lista inteira:

print("Imprimindo a lista de uma vez:")
print(nomes)

Ou então imprimir separadamente cada item da lista:

print("\nImprimindo item por item:")

for i in range(qtd):
    print(nomes[i], end=" ")


print("\n\nImprimindo item por item:")

for i in range(0, qtd - 2):
    print(nomes[i], end=", ")
print(nomes[qtd - 2], "e", nomes[qtd - 1])
 
print("\n\nImprimindo item por item:")
print(nomes[0], end="")
for i in range(1, qtd - 1):
    print(",", nomes[i], end="")
print(" e", nomes[qtd - 1])

Preenchendo listas

Para preencher uma lista com valores digitados pelo usuário:

N = 5
nums = []

for i in range(N):
    num = int(input(f"Informe o {i + 1}º número: "))
    nums.append(num)

print("Você digitou os seguintes números:")
for i in range(N):
    print(nums[i], end=" ")

Modificando listas

Você pode modificar os itens de uma lista:

qtd = 10
nomes = ["maria", "ana", "francisca", "antonia", "adriana",\
         "jose", "joao", "antonio", "francisco", "luiz"]

print("Nomes:", nomes)

print("\nAlterando...\n")

for i in range(qtd):
    nomes[i] = nomes[i].upper()

print("Nomes alterados:", nomes)

Visão geral

  • Uma lista é como se fosse várias variáveis com o mesmo nome (o nome da lista).
  • Você distingue entre as "variáveis" da lista (seus itens) com um índice numérico (posição do item na lista).
  • O primeiro item da lista está no índice 0 e o restante conta a partir daí.
  • Você acessa/modifica um item da lista pelo seu índice.
  • Os índices da lista ficam dentro dos colchetes.
  • Atenção: não coloque um índice maior do que o comprimento da lista para não dar erro!
  • Diferentemente de arrays convencionais, que possuem um comprimento fixo definido na sua inicialização, listas em Python possuem comprimentos dinâmicos. Elas podem crescer ou diminuir seu comprimento conforme itens são adicionados ou excluídos.

O comprimento da lista

Para saber quantos itens há em uma lista, use a função len, abreviação de length (comprimento, em inglês):

frutas = ["pequi", "pera", "pêssego", "pitanga", "pinha", "pitaia"]
legumes = ["caxi", "cebola", "cenoura", "chuchu"]

qtd_frutas = len(frutas)
qtd_legumes = len(legumes)

print(f"\nEu tenho {qtd_frutas} frutas:")
for i in range(qtd_frutas):
    print(frutas[i], end=" ")

print(f"\n\nTambém tenho {qtd_legumes} legumes:")
for i in range(qtd_legumes):
    print(legumes[i], end=" ")

A instrução for-each

Listas em Python fazem parte de uma classe de dados chamada de iteráveis. Iterar uma lista significa percorrer cada item da lista.

A instrução for-each, permite iterar pelos itens de uma classe iterável sem utilizar um contador.

Sua vantagem é que Evita a possibilidade de ultrapassar o limite da classe iterável.

A sintaxe de uma instrução for-each é:

for it in iterable:
    instruções...

Onde it é o item atual da lista iterable.

Exemplo:

array = [87, 68, 94, 100, 83, 78, 85, 91, 76, 87]

total = 0

for number in array:
    total += number

print("Total of array items:", total)

A instrução for-each não pode ser utilizada para modificar itens em um array. Se precisar modificar itens, utilize a tradicional instrução for.

Observe no seguinte exemplo que nada adianta modificar a variável e achando que os itens da lista serão afetados:

Quando você faz isso… … Na verdade significa isso 
arr = [4, 3, 2, 1]

for e in arr:
    e = e * 2

print(arr)
 
arr = [4, 3, 2, 1]

for i in range(len(arr)):
    e = arr[i]  # `e` é uma cópia de `arr[i]`
    e = e * 2
print(arr)

Exercícios

  1. Ler uma lista de n itens. Escrever os n itens normalmente em uma linha e do último para o primeiro em outra linha.
  2. Ler uma lista de n números e dois índices x e y. Escrever a soma dos itens nos índices x e y.
  3. Ler uma lista de n números. Escrever os itens dos índices ímpares da lista.
  4. Ler uma lista de n números. Escrever os itens ímpares da lista.
  5. Seja a função $y = -2x + 10$. Ler um número inteiro $n$ e preencher uma lista com os valores de $y$ para todos os valores de $x$ no intervalo de números inteiros entre $1$ e $n$. Escrever a lista resultante.
  6. Ler uma lista de n itens e dois índices x e y. Permutar os itens de x e y, e escrever a lista resultante.
  7. Ler uma lista de n itens e um valor x. Escrever o primeiro índice em que x se encontra na lista. Caso x não se encontre, escrever "Não encontrado!".
  8. Ler uma lista de n números. Escrever o índice do maior item. Caso haja empate, escrever o de menor índice.
  9. Ler uma lista de n itens, seguida por uma sequência de índices que se encerra com um índice inválido. Para cada índice, escrever seu item.
  10. Ler uma lista de n itens, seguida por uma sequência de itens que se encerra com um item que não existe na lista. Para cada item, escrever seu índice.
  11. Ler um natural $n$, indicando a capacidade máxima de itens em uma lista, seguido por uma série de duplas $i$ e $x$. Cada dupla representa um índice $i$ e seu respectivo item $x$ na lista. Encerrar a leitura ao encontrar um índice inválido. Caso não seja fornecido um determinado índice, considerar seu item como zero. Escrever a lista resultante.
  12. Ler uma lista de n itens, seguida por uma sequência de itens que se encerra com um item ausente na lista. Para cada item, substituir sua primeira ocorrência na lista por zero. Escrever a lista resultante.
  13. Dada uma lista de n itens, há itens repetidos na lista?
  14. Ler uma lista de n números e um número m. Escrever os números da lista que são menores que m em uma linha, e os maiores que m em outra linha.
  15. Ler uma lista de n números. Escrever os números menores que a média aritmética em uma linha e os maiores em outra linha.
  16. Ler uma lista de n itens. Permutar os itens da primeira metade com os da segunda metade da lista. Caso n seja ímpar, o item do meio fica na mesma posição. Escrever a lista resultante.
  17. Ler uma lista de n números. Compactar a lista, movendo todos os zeros para o final. Escrever a lista resultante. Exemplo, dado a lista 13 0 90 0 0 57 0 99 0 40 escrever 13 90 57 99 40 0 0 0 0 0.
  18. Ler duas listas ordenadas de n e m números, respectivamente. Escrever uma terceira lista ordenada contendo os números das outras duas listas. (Obs.: Essa operação faz parte de um importante algoritmo de ordenação chamado mergesort).
  19. Ler uma lista de n números e um número m. Reorganizar a lista de modo que seus números menores que m fiquem à esquerda de seus números maiores que m. Escrever a lista resultante. (Obs.: Essa operação faz parte de um importante algoritmo de ordenação chamado quicksort).
  20. Ler um natural n seguido por n números. Escrever os n números em ordem crescente.
