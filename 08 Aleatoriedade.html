Aleatoriedade

Sorte é o encontro do preparo com a oportunidade.

Considere o seguinte trecho de código para gerar números aleatórios. Preste atenção aos comentários para entender o que está se passando.

import random

# sorteia um float no intervalo [0,1)
sorteado = random.random()
print(sorteado)

# transforma em um float entre [0,10)
sorteado = sorteado * 10
print(sorteado)

# descarta a parte decimal
int_sorteado = int(sorteado)
print(int_sorteado)

O Python é uma linguagem extensível, permitindo a incorporação de diversos códigos desenvolvidos por terceiros. Ao instalar o Python, uma biblioteca padrão de módulos já está disponível para uso. Módulos são arquivos que contém diversas funções e outros recursos para você agregar ao seu código conforme necessidade. Para usar os recursos de algum módulo, é necessário inicialmente importá-lo em seu programa por meio do comando import, seguido pelo nome correspondente ao módulo desejado. Neste caso, importamos (carregamos) o módulo random, que disponibiliza uma gama de funções voltadas à geração de números aleatórios. Você importa o módulo apenas uma vez no script que pretende utilizar, tipicamente no início do código.

Sempre que utilizar uma função de um módulo, preceda o nome da função pelo nome do módulo seguido por um ponto, como em random.random(). Neste caso, o primeiro random se refere ao módulo e o segundo random se refere à função. Observe que sempre que usarmos funções, elas precisarão estar entre aspas. A notação matemática [0,1) significa o intervalo de números entre o 0 inclusivo e o 1 exclusivo, isto é, números maiores ou iguais a 0 e menores que 1.

O trecho de código anterior gerou um inteiro aleatório entre [0,10). Podemos simplificar em apenas uma instrução a geração de um inteiro aleatório entre [0,n):

import random

n = 20
sorteado = int(random.random()*n)
print(sorteado)

Se você não quiser começar do zero, pode usar um valor de deslocamento. Por exemplo, para gerar um número aleatório entre [10,20], você pode fazer:

primeiro = 10
ultimo = 20
quantidade = ultimo - primeiro + 1
sorteado = primeiro + int(random.random() * quantidade)

Você também pode gerar um número aleatório dentro de um intervalo discreto. Por exemplo, para gerar um múltiplo de 5 entre [50,100], você pode fazer:

primeiro = 50
ultimo = 100
diferenca = 5
quantidade = (ultimo - primeiro) / diferenca + 1
sorteado = primeiro + diferenca * int(random.random() * quantidade)

Essa é a fórmula geral para gerar números inteiros. Você pode ajustá-la conforme sua necessidade. Apenas altere as variáveis primeiro, ultimo e diferenca. Elas representam, respectivamente, o primeiro número inclusivo do intervalo, o último número inclusivo do intervalo, e a diferença entre dois números consecutivos do intervalo.

Semente

As funções presentes no módulo random utilizam uma semente para gerar uma sequência de números aleatórios. Isso faz elas se comportarem de modo determinístico, isto é, para uma mesma semente, a mesma sequência de números aleatórios será gerada. Isso implica que esses números são considerados "pseudoaleatórios", uma vez que a aleatoriedade é comprometida se a semente utilizada for conhecida, permitindo a previsão dos valores que serão produzidos.

Essa geração de números inteiros com aparência de aleatoriedade tem diversas aplicações científicas, como simulações de sistemas complexos. A característica determinística na geração de tais números muitas vezes é vantajosa, pois a capacidade de reproduzir um experimento científico desempenha um papel crucial na validação do estudo.

Caso você necessite de geração de números verdadeiramente não determinística, o módulo secrets está disponível para esse propósito, seguindo um padrão de uso semelhante ao módulo random.

Se não informar a semente, o Python utiliza o horário atual como semente. Para informar a semente:

import random
        
random.seed(2)

print(random.random())
print(random.random())
print(random.random())
print(random.random())
print(random.random())

Execute várias vezes esse programa com e sem a instrução de seed para entender seu comportamento.

Exercícios

  1. Dado um inteiro como semente, gerar três inteiros aleatórios.
  2. Dado um inteiro $n$, gerar três inteiros aleatórios entre 0 e $n$.
  3. Dado um inteiro $n$, gerar três inteiros aleatórios entre $-n$ e $n$.
  4. Dados dois inteiros $m$ e $n$, gerar três inteiros aleatórios entre $m$ e $n$.
  5. Dados dois inteiros $m$ e $n$, gerar três inteiros aleatórios múltiplos de $m$ entre $0$ e $n$.
  6. Gerar dois inteiros aleatórios cuja soma esteja entre [0 e 100).
  7. Gerar uma letra aleatória entre 'a' e 'z'.
  8. Gerar uma senha aleatória de 8 dígitos com letras maiúsculas, minúsculas e inteiros.
  9. Gerar três decimais aleatórios entre [-100 e 100], com até duas casas decimais.
  10. Ler um número inteiro e escrever um valor aleatório com base nesse número: uma letra maiúscula se o usuário inseriu 1, uma letra minúscula se o usuário inseriu 2 e um dígito se o usuário inseriu qualquer valor que não seja 1 ou 2.
