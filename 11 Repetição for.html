Repetição for

A roda já deu uma volta completa. — William Shakespeare

Principios da repetição controlada por contador

Na repetição controlada por contador, sabemos de antemão quantas vezes o corpo da repetição será executado. O contador começa de um valor inicial e é atualizado a cada iteração (ciclo) até alcançar um valor final.

Em vez de usarmos o while para repetição por contador, podemos usar o for:

inicio = 1
fim = 11
print("Contando até 10...")

cont = inicio
while cont < fim:
    print(cont)
    cont += 1

print("Fim!")
print("Valor final:", cont)
 
inicio = 1
fim = 11
print("Contando até 10...")


for cont in range(inicio, fim):
    print(cont)


print("Fim!")
print("Valor final:", cont)

No Python, definimos um intervalo de valores inteiros para o loop for através da função range. Note que este intervalo é semiaberto, ou seja, o valor inicial está incluído, enquanto o valor final não está. Observação: a função range aceita apenas números inteiros.

Observe que no for o contador termina com um valor a menos que o final do intervalo, enquanto que no while o contador termina com o valor final do intervalo, apesar de ambos executarem o corpo da repetição a mesma quantidade de vezes.

Diferenças e semelhanças entre for e while

Tudo o que dá para fazer usando while também dá usando for e vice-versa. Porém dependendo da situação, um pode ser mais adequado que outro. Em geral, as instruções for são utilizadas para repetição controlada por contador: quando sabemos quantas vezes queremos repetir determinado trecho de código… E as instruções while, para repetição controlada por sentinela (flag): quando não sabemos o número de vezes que queremos repetir determinado trecho de código.

i = 1
soma = 0
while i < 5:
    print("+", i, end=" ")
    soma += i
    i += 1
print("= ", soma)
 

soma = 0
for i in range(1, 5):
    print("+", i, end=" ")
    soma += i

print("= ", soma)

Mais possibilidades com o for

Você pode omitir o valor inicial se quiser começar do zero:

print("Contando do 0 até o 9...")
for cont in range(10):
    print(cont)

Você pode especificar o tamanho do teu "passo" na contagem:

print("Contagem de 5 em 5...")
inicio = 0
fim = 50
passo = 5
for cont in range(inicio, fim, passo):
    print(cont)

Se o teu passo for negativo e o valor inicial for maior que o valor final, a contagem será regressiva:

print("Contagem regressiva...")
for cont in range(10, 0, -1):
    print(cont)

Assim como qualquer estrutura que possui corpo, se este for composto por apenas uma linha de código, pode ir na mesma linha que a estrutura:

print("Contagem regressiva...")
for cont in range(10, 0, -1): print(cont)

Assim como qualquer estrutura que possui corpo, não se esqueça do dois-pontos separando a declaração da estrutura do seu corpo!

Observação: Assim como no while, o for pode nunca ser executado!

print("Valendo!")
for cont in range(0, 10, -1):
    print(cont)
print("Fim!")

Mais exemplos de for

Contando de 1 a 100 em incrementos de 1:

for i in range(1, 101): print(i)

Contando de 100 a 1 em decrementos de 1:

for i in range(100, 0, -1): print(i)

Contando de 7 a 77 em incrementos de 7:

for i in range(7, 78, 7): print(i)

Contando de 20 a 2 em decrementos de 2:

for i in range(20, 1, -2): print(i)

Incrementando entre os valores 2, 5, 8, 11, 14, 17, 20:

for i in range(2, 21, 3): print(i)

Decrementando entre os valores 99, 88, 77, 66, 55, 44, 33, 22, 11, 0:

for i in range(99, -1, -11): print(i)

Laços aninhados

Seja estruturas de seleção ou de repetição, podemos colocá-las umas dentro das outras. Veja este exemplo de um for dentro de outro for.

Veja esse exemplo de if-else dentro de for. Qual a saída?

i = 12
for j in range(1, i + 1):
    print("", j, end="")
    if j % 5 == 0 or j % 3 == 0:
        print(" A", end="")
    if i % j == 0:
        print(" B", end="")
    else:
        print(" C", end="")

Vamos examinar um exemplo de laço for simples:

i = 1
print("(", end=" ")
for j in range(0, 5):
    print(j, end=" ")
print(")")

A linha 4 é executada 5 vezes. Agora, vamos colocar esse laço dentro de outro laço:

for i in range(0, 3):
    print("(", end=" ")
    for j in range(0, 5):
        print(j, end=" ")
    print(")")

Neste novo código, devido ao laço da linha 1, as linhas 2 a 5 são executadas 3 vezes. A linha 4, que já era executada 5 vezes no laço interno, agora é executada um total de 3 * 5 = 15 vezes!

O laço interno repete a mesma quantidade de vezes a cada ciclo do laço externo. Mas poderíamos alterar esse comportamento:

for i in range(0, 3):
    print("(", end=" ")
    for j in range(i, 5):
        print(j, end=" ")
    print(")")

Qual a saída do seguinte código?

print("Início")
for externo in range(1, 4):
    print(">", end=" ")
    for interno in range(1, 6):
        print(externo, interno, sep="", end=" ")
    print()  # Adiciona uma quebra de linha após a conclusão de cada linha interna
print("Fim")

Qual a saída do seguinte código?

for i in range(0, 3):
    for j in range(0, 4):
        for k in range(0, 2):
            print("3", end="")
        print("2", end="")
    print("1", end="")
