UVa 12015 - Google está com Sorte

O Google é um dos motores de busca na Internet mais famosos, que desenvolve uma variedade de serviços e produtos online. Em seu site de busca, um interessante botão, “Estou com Sorte”, chama a atenção. Esse recurso permite que o usuário pule a página de resultados de busca e vá diretamente para a primeira página classificada. Incrível! Isso economiza muito tempo.

A questão é: quando alguém digita algumas palavras-chave e pressiona o botão “Estou com Sorte”, qual página da web aparecerá? O Google usa métodos avançados para determinar isso. Neste problema simplificado, consideramos apenas que o Google atribui uma relevância numérica a cada página da web. A página mais relacionada será escolhida. Em caso de empate, todas as páginas com a maior relevância podem ser escolhidas.

Sua tarefa é simples: dada uma lista de 10 páginas da web e suas relevâncias, selecione todas as possíveis candidatas que serão apresentadas ao usuário ao clicar em “Estou com Sorte”.

Entrada

A entrada contém múltiplos casos de teste. O número de casos de teste T está na primeira linha do arquivo de entrada. Para cada caso de teste, há 10 linhas, cada uma descrevendo uma página da web e sua relevância. Cada linha contém uma string de caracteres sem espaços representando a URL da página e um inteiro Vi indicando a relevância dessa página (1 ≤ Vi ≤ 100). O comprimento da URL está entre 1 e 100 caracteres, inclusive.

Saída

Para cada caso de teste, imprima várias linhas contendo as URLs das páginas da web que podem ser escolhidas. A ordem das URLs deve ser a mesma da entrada. Veja o exemplo de saída para mais informações sobre o formato da saída.
