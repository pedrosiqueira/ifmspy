UVa 12770 Palinagrama

Um palíndromo é uma palavra que é escrita da mesma forma de trás para frente ou de frente para trás. Exemplos são "level" e "madam". Um anagrama é uma palavra formada a partir de outra palavra apenas reorganizando os caracteres, como de "made" para "dame". Para este problema, anagramas e palíndromos podem não ser palavras válidas em inglês. Por exemplo, "daamm" é um anagrama de "madam" e "amdma" também é um palíndromo, embora não seja uma palavra em inglês. "daamm" também é chamado de palinagrama porque é o anagrama de um palíndromo. Qualquer palíndromo também é um palinagrama, mas o inverso nem sempre é verdade. Alguns outros exemplos de palinagramas são "aabbcc", "aaaaa", etc. Dada uma string, você precisará imprimir a string que deve ser anexada a ela para torná-la um palinagrama.

Entrada

O arquivo de entrada contém no máximo 6000 casos de teste. A descrição de cada caso de teste é dada abaixo.

Cada caso consiste em uma única string S de comprimento L (1 ≤ L ≤ 500). Esta string contém apenas letras minúsculas do alfabeto inglês (‘a’ a ‘z’).

A entrada é encerrada por uma linha contendo um único caractere cerquilha ('#'). Esta linha não deve ser processada.

Saída

Para cada linha de entrada, produza uma linha de saída. A linha contém a string que precisa ser anexada à string dada para torná-la um palinagrama (pode ser uma string vazia também). Se houver mais de uma solução, imprima a mais curta. Se ainda houver empate, imprima a menor em ordem lexicográfica.

Exemplo de Entrada/Saída

Entrada Saída 
ddc
aaab
#
 
a
