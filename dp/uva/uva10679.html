UVa 10679 - Eu Amo Strings!!!

Hmmmmmm... strings de novo :) Então, deve ser uma tarefa fácil para você. Bem, você recebe uma string S com comprimento de no máximo 100.000 caracteres (contendo apenas letras de ‘a’ a ‘z’ e de ‘A’ a ‘Z’). Em seguida, há q consultas (q < 1000), onde cada consulta contém uma string T de comprimento máximo 1.000 (também contendo apenas letras de ‘a’ a ‘z’ e de ‘A’ a ‘Z’).

Seu objetivo é determinar se T é uma sub-string de S.

Entrada

A primeira linha contém um inteiro k (k < 10), indicando o número de casos de teste. Cada caso de teste começa com a string em uma linhaS. A linha seguinte contém um inteiro q, indicando o número de consultas. Cada uma das próximas q linhas contêm uma string T.

Saída

Para cada consulta, imprima 'y' se T for uma sub-string de S, ou 'n' caso contrário. Cada resposta deve estar em uma nova linha.
