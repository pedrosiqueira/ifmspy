UVa 10038 - Jolly Jumpers

Uma sequência de n > 0 inteiros é chamada de "jolly jumper" se os valores absolutos das diferenças entre elementos sucessivos assumirem todos os valores de 1 até n - 1. Por exemplo, a sequência 1 4 2 3 é um jolly jumper, pois as diferenças absolutas são 3, 2 e 1, respectivamente. A definição implica que qualquer sequência de um único inteiro é um jolly jumper.

Entrada

Cada linha de entrada contém um inteiro n (onde n ≤ 3000), seguido por n inteiros representando a sequência.

Saída

Para cada linha de entrada, gere uma linha de saída dizendo Jolly se a sequência for um jolly jumper ou Not jolly caso contrário.

Exemplo de Entrada/Saída

Entrada Saída 
4 1 4 2 3
5 1 4 2 -1 6
 
Jolly
Not jolly

Dicas

Para converter uma string line de números separados por espaço em uma lista numbers de números, em Python, use: numbers = list(map(int, line.split())).

Por exemplo, esse código converte "5 1 4 2 -1 6" em [5, 1, 4, 2, -1, 6].

Uma possível solução para esse problema é criar uma lista de booleanos de comprimento n e marcar como True os índices correspondentes aos valores das diferenças absolutas entre elementos consecutivos da sequência.
