UVa 10252 - Permutação Comum

Dadas duas strings de letras minúsculas, a e b, imprima a maior string x de letras minúsculas tal que existe uma permutação de x que seja uma subsequência de a e uma permutação de x que seja uma subsequência de b.

Entrada

O arquivo de entrada contém várias linhas de entrada. Linhas consecutivas (pares de linhas) formam um conjunto de entrada: a linha 1 e 2 formam o primeiro conjunto, a linha 3 e 4 o segundo, e assim por diante. A primeira linha do par contém a string a e a segunda linha contém b. Cada string está em uma linha separada e contém no máximo 1000 letras minúsculas.

Saída

Para cada conjunto de entrada, imprima uma linha contendo x. Se houver várias strings x que satisfaçam o critério, escolha a primeira em ordem alfabética.
