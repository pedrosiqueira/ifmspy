UVa 10324 - Zeros e Uns

Dada uma string de 0’s e 1’s com até 1.000.000 de caracteres e índices i e j, você deve responder se todos os caracteres entre a posição min(i, j) e a posição max(i, j) (inclusiva) são iguais.

Entrada

Há múltiplos casos de teste na entrada. A primeira linha de cada caso fornece uma string de 0’s e 1’s. A próxima linha contém um número inteiro positivo n, que indica o número de consultas para esse caso. As próximas n linhas contêm consultas, uma por linha. Cada consulta é composta por dois inteiros não-negativos i e j. Para cada consulta, você deve imprimir ‘Yes’ se todos os caracteres da string entre as posições min(i, j) e max(i, j) forem iguais, e ‘No’ caso contrário.

Saída

Cada caso de saída deve começar com um cabeçalho conforme mostrado no exemplo abaixo. A entrada termina com uma linha vazia contendo apenas o caractere de nova linha, e essa linha não deve ser processada. A entrada também pode terminar com o fim do arquivo, então mantenha a verificação para ambas as condições.
