UVa 11679 - Sub-prime

A crise econômica mais recente foi causada em parte pela forma como os bancos concediam empréstimos a pessoas incapazes de pagá-los e revendiam esses empréstimos a outros bancos como debêntures. Obviamente, quando as pessoas deixaram de pagar seus empréstimos, todo o sistema colapsou.

A crise foi tão profunda que afetou países de todo o mundo, incluindo Nlogônia, onde o respeitado primeiro-ministro Man Dashuva ordenou que o presidente do Banco Central encontrasse uma solução. Ele teve uma ideia brilhante: se todos os bancos conseguissem liquidar suas debêntures apenas com suas próprias reservas monetárias, todos os bancos sobreviveriam e a crise seria evitada.

No entanto, com o elevado número de debêntures e bancos envolvidos, essa era uma tarefa extremamente complicada, então ele pediu sua ajuda para escrever um programa que, dado o conjunto de bancos e debêntures emitidas por eles, determine se é possível que todos os bancos paguem suas dívidas usando apenas suas reservas monetárias e créditos.

Entrada

A entrada consiste em vários casos de teste. A primeira linha de cada caso de teste contém dois inteiros B e N, indicando o número de bancos (1 ≤ B ≤ 20) e o número de debêntures emitidas pelos bancos (1 ≤ N ≤ 20). Os bancos são identificados por inteiros entre 1 e B.

A segunda linha contém B inteiros Ri separados por espaços, indicando as reservas monetárias de cada um dos B bancos (0 ≤ Ri ≤ 104, para 1 ≤ i ≤ B).

As N linhas seguintes contêm cada uma três inteiros separados por espaços: um inteiro D, indicando o banco devedor (1 ≤ D ≤ B), um inteiro C, indicando o banco credor (1 ≤ C ≤ B e D ≠ C), e um inteiro V, indicando o valor da debênture (1 ≤ V ≤ 104).

O fim da entrada é indicado por uma linha contendo apenas dois zeros, separados por espaços.

Saída

Para cada caso de teste, o programa deve imprimir uma única linha contendo um único caractere: ‘S’, se for possível liquidar todas as debêntures sem intervenção do Banco Central de Nlogônia, ou ‘N’ se uma intervenção for necessária.
