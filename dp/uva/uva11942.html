UVa 11942 - Sequenciamento dos Lenhadores

Mais uma história de lenhadores? Vamos ver…

Os lenhadores são trabalhadores rudes e barbados, enquanto os capatazes tendem a ser mandões e de mentalidade simples. Os capatazes gostam de importunar os lenhadores, fazendo-os alinhar-se em grupos de dez, ordenados pelo comprimento de suas barbas. Os lenhadores, tendo alturas físicas variadas, mudam suas posições para confundir os capatazes. Assim, os capatazes precisam realmente medir as barbas em centímetros para ver se todos estão alinhados corretamente.

Sua tarefa é escrever um programa para ajudar os capatazes a determinar se os lenhadores estão alinhados corretamente, seja da barba mais curta para a mais longa, ou da mais longa para a mais curta.

Entrada

A entrada começa com uma linha contendo um único inteiro N, 0 < N < 20, que é o número de grupos a serem processados. A seguir, há N linhas, cada uma contendo dez inteiros positivos distintos, menores que 100.

Saída

A saída deve conter uma linha de título e, em seguida, uma linha para cada conjunto de comprimentos de barbas. Consulte o exemplo de saída para detalhes de capitalização e pontuação.
