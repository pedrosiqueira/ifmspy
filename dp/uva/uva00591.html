UVa 591 - Box of Bricks

O pequeno Bob gosta de brincar com sua caixa de blocos. Ele empilha os blocos e constrói pilhas de diferentes alturas. “Olha, eu construí uma parede!”, ele diz à sua irmã mais velha, Alice. “Não, você deveria fazer todas as pilhas terem a mesma altura. Então você teria uma parede de verdade.”, ela responde. Após pensar um pouco, Bob percebe que ela está certa. Então, ele decide reorganizar os blocos, um por um, para que todas as pilhas fiquem com a mesma altura. Mas, como Bob é preguiçoso, ele quer fazer isso com o menor número possível de movimentos. Você pode ajudar?

Entrada

A entrada consiste em vários conjuntos de dados. Cada conjunto começa com uma linha contendo o número n de pilhas que Bob construiu. A próxima linha contém n números, representando as alturas hi das n pilhas. Você pode assumir que 1 ≤ n ≤ 50 e 1 ≤ hi ≤ 100.

O número total de blocos será divisível pelo número de pilhas. Assim, sempre será possível reorganizar os blocos de forma que todas as pilhas fiquem com a mesma altura.

A entrada é terminada por um conjunto começando com n = 0. Este conjunto não deve ser processado.

Saída

Para cada conjunto, primeiro imprima o número do conjunto, como mostrado no exemplo de saída. Em seguida, imprima a linha The minimum number of moves is k., onde k é o número mínimo de blocos que devem ser movidos para que todas as pilhas fiquem com a mesma altura.

Imprima uma linha em branco após cada conjunto.

Exemplo de Entrada/Saída

Entrada Saída 
6
5 2 4 1 7 5
0
 
Set #1
The minimum number of moves is 5.
