UVa 01585 Pontuação

Há um resultado de uma prova objetiva, como "OOXXOXXOOO". Um ‘O’ significa uma resposta correta de uma questão, e um ‘X’ significa uma resposta errada. A pontuação de cada questão desta prova é calculada com base nela mesma e nos ‘O’s consecutivos imediatamente anteriores, apenas quando a resposta estiver correta. Por exemplo, a pontuação da 10ª questão é 3, que é obtida pela própria questão e pelos dois ‘O’s consecutivos imediatamente anteriores.

Portanto, a pontuação de "OOXXOXXOOO" é 10, calculada como "1+2+0+0+1+0+0+1+2+3".

Você deve escrever um programa que calcule as pontuações dos resultados das provas.

Entrada

Seu programa deve ler a entrada padrão. A entrada consiste em T casos de teste. O número de casos de teste T é dado na primeira linha da entrada. Cada caso de teste começa com uma linha contendo uma string composta por ‘O’ e ‘X’, e o comprimento da string é maior que 0 e menor que 80. Não há espaços entre os ‘O’s e ‘X’s.

Saída

Seu programa deve escrever na saída padrão. Imprima exatamente uma linha para cada caso de teste. A linha deve conter a pontuação do caso de teste.

Exemplo de entrada/saída

Entrada Saída 
5
OOXXOXXOOO
OOXXOOXXOO
OXOXOXOXOXOXOX
OOOOOOOOOO
OOOOXOOOOXOOOOX
